# Multi-Model ROI Localization and Medical Image Classification

## Project Overview

This project focuses on Region of Interest (ROI) Localization and Medical Image Classification using Deep Learning techniques. The system automatically identifies important regions in medical images and classifies them into different disease categories.

## Objectives

- Detect Region of Interest (ROI) from medical images.
- Improve diagnostic accuracy using AI.
- Perform image preprocessing and feature extraction.
- Classify medical images using Deep Learning models.
